Vision Language Action Models#

ManiSkill supports evaluating and pretraining vision language action models. Currently the following VLAs have been tested via the ManiSkill framework:

  • Octo

  • RDT-1B

  • RT-x

RDT-1B uses some of the ManiSkill demonstrations for pretraining data and evaluates by fine-tuning on some demonstrations on various ManiSkill tasks, see their README for more details.

Octo and RT series of models are evaluated through various real2sim environments as part of the SIMPLER project, see their README for details on how to run the evaluation setup.
